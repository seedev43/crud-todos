# ToDo App

Aplikasi Todo App dengan teknologi MERN (MongoDB, Express, React, NodeJS) Stack.

## Instalasi

### Untuk API Backend

`cd backend`

`npm install`

`npm run dev`

buka `http://localhost:5000` di browser

### Untuk Frontend

`cd frontend`

`npm install`

`npm run dev`

buka `http://localhost:5173` di browser
